
Base de datos
*************
Este proyecto final consiste en una base de datos de una tienda de electromesticos 
en el que tenemos 5 opciones :


En la primera opcion :
	**se ingresan los datos de los clientes , aqui van 
		/-nombre
		/-direcion
		/-edad
		/-cantidad de dinero que poseen
		/-codigo(generado automaticamenteen el archivo .txt)

En la segunda opcion :
	**se ingresan los datos del producto mediante la funcion 'setdata'
	  encargada de almacenar los datos en un archivo txt

En la tercera opcion :
	**se visualizaran los diferentes productos de cada categoria

En la cuarta opcion :
	**se pedira ingresar el codigo del producto , para asi poder observar los datos del producto

en la quinta opcion : 
	**se realizara la compra del producto ,pero antes pidiendo el codigo del cliente para su 
	  respectiva verificacion.
